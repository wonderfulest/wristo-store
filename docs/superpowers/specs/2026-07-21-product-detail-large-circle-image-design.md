# 产品详情页大圆产品图设计

## 目标

将 Store 产品详情页左侧当前的“方形舞台内嵌较小圆形产品图”调整为单个更大的正圆产品图，让表盘主体成为视觉焦点。

## 改动范围

- 仅调整 `ProductImageGallery.vue` 的主图舞台样式。
- 移除主图外围可见的方形背景、边框和方形阴影。
- 圆形主图占满图库可用宽度，始终保持 `1:1` 和 `border-radius: 50%`。
- 保留点击全屏预览、键盘切换、触摸滑动、左右切换按钮、缩略图和图片加载失败处理。
- 保持产品购买信息区、详情内容区、接口和数据结构不变。

## 响应式行为

桌面端圆形主图随左侧图片列放大，最大尺寸仍由现有图片列宽度控制。移动端继续使用产品详情页已有的 `280px` 和 `240px` 图片列宽度，因此圆形不会溢出屏幕。

## 验证

- 运行现有产品图库测试，确认选择、切换和失败回退逻辑不受影响。
- 运行 `npm run build`，确认 Vue、TypeScript 和样式构建通过。
- 检查桌面和移动断点下主图均为无外层方框的大正圆。

# 首页首屏海报聚焦设计

## 目标

让国际站首页首屏以右侧 3:4 表盘海报为主要视觉，减少左侧重复营销信息，使用户能在一次扫视中理解“丰富表盘选择”并进入浏览页。

## 范围

- 仅调整 `HomeBanner.vue` 中 Store 首屏轮播项。
- Studio 轮播项保持现有内容、按钮、指标和图片布局。
- 仅改默认英文标题与描述；其他语言继续使用现有翻译。

## 文案

- 标题：`Discover Your Next Watch Face.`
- 描述：`5000+ designs for every style.`
- Store 首屏仅显示主按钮，按钮文案改为：`Explore Watch Faces`
- 隐藏 Store 首屏的 `Enter code`、`View bundles` 和三项指标。

## 布局

- Store 首屏使用“窄文案、宽海报”双栏比例，右栏成为视觉主体。
- 桌面端右侧舞台最大宽度约 `600px`，海报最大展示约 `510 × 680px`，保持 3:4 比例且不裁切。
- 中等屏幕改为单栏后，海报仍完整展示；移动端宽度限制在内容区内，不产生横向滚动。
- 海报模式继续隐藏舞台编号、光晕和旧的 `Artist series / Battery-aware` 角标。

## 实现方式

- 在 Store 轮播项增加精简展示标记，由模板按标记隐藏次要操作和指标。
- 通过 Store 主题类和海报类调整网格比例、舞台尺寸及响应式规则。
- 不修改路由、数据请求、轮播计时或 Studio 权限逻辑。

## 验证

- 运行 `npm run build`，覆盖 TypeScript、Vite 生产构建和 SEO 预渲染。
- 运行 `git diff --check`。
- 若本机浏览器策略允许访问开发服务，检查桌面与移动端首屏；否则明确记录该环境限制。

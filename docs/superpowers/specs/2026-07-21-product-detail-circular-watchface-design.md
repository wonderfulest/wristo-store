# 产品详情页正圆表盘展示设计

## 目标

优化 Store 产品详情页左侧图片画廊，让主图只展示表盘的正圆区域，并通过克制的层次和光影强化表盘质感。下方缩略图继续展示完整原图。

## 改动范围

- 仅修改 `ProductImageGallery.vue` 的主图舞台视觉。
- 保留现有方形画廊舞台和自适应正方形布局。
- 主图中央增加正圆视口，图片在圆内完整适配并裁掉圆外内容。
- 正圆增加细描边、内高光和柔和阴影，与 Store 当前青绿色视觉体系保持一致。
- 适当弱化舞台的图片模糊背景，使视觉焦点集中在表盘主体。
- 移动端缩小圆形主体占比并保留安全留白，避免贴边。

## 保持不变

- 下方缩略图继续显示完整图片，不使用圆形裁切。
- 图片切换、键盘操作、触摸滑动、全屏预览和错误回退逻辑不变。
- 下载入口、编辑态排序和图片管理功能不变。
- 产品详情页右侧购买、价格、设备兼容和详情内容不变。

## 实现方式

在现有主图元素外增加一个只负责视觉裁切的圆形容器。容器维持 `aspect-ratio: 1` 和 `border-radius: 50%`，内部图片继续使用 `contain`，同时由容器隐藏圆外内容。圆形尺寸通过响应式 CSS 控制，不新增业务状态或数据接口。

全屏预览仍由现有 Element Plus 图片组件触发，预览原始完整图片，不受页面圆形蒙版影响。

## 验证标准

- 桌面端主图只显示正圆范围，圆形居中且四周留白均衡。
- 移动端主图不贴边、不变形。
- 缩略图仍能看到完整原图。
- 点击主图可正常打开完整图片预览。
- 多图切换、键盘方向键、移动端滑动和图片失败回退保持可用。
- `npm run build` 通过。

# Wristo Brand Assets

## Core Idea

The Wristo mark is a circular watch-face orbit built for small social avatars and site usage. The teal outer circle represents the smartwatch surface, the white dial keeps the mark readable, and the amber hand adds a clear time-design signal.

## Files

- `wristo-logo-primary.svg` - light-background logo lockup for website header, footer, and documents.
- `wristo-logo-primary-dark.svg` - dark-background logo lockup.
- `wristo-mark.svg` - standalone app icon, favicon source, watermark, and small UI mark.
- `social-avatar.svg` - general 1:1 avatar source.
- `instagram-profile.svg` - Instagram profile image source.
- `pinterest-profile.svg` - Pinterest profile image source.
- `facebook-cover.svg` - Facebook cover source, 820 x 312.
- `youtube-cover.svg` - YouTube channel cover source, 2560 x 1440 with the main content centered in the safe area.

Matching `.png` exports are included beside each SVG for direct upload or quick preview.

## Color Tokens

- Brand teal: `#0F6B68`
- Brand teal dark: `#071F1E`
- Ink: `#111827`
- Canvas: `#FBFDFC`
- Accent amber: `#F59E0B`
- Soft teal: `#DFF5F1`

## Typography

Use `DM Sans` for the wordmark and supporting text. The website already loads DM Sans, so the SVGs match the current storefront typography. If converting to PNG for social upload, open the SVG in a browser after fonts load or export from a design tool with DM Sans installed.

## Usage Rules

- Keep the mark inside a square or circle for social media.
- Use `wristo-logo-primary.svg` on light surfaces and `wristo-logo-primary-dark.svg` on dark surfaces.
- Do not recolor the amber hand; it is the time cue.
- Keep at least 12.5% clear space around the standalone mark.
- Avoid placing the full wordmark below 220 px wide; use `wristo-mark.svg` instead.

## Suggested Social Uploads

- Facebook profile: use `social-avatar.png`, 1080 x 1080.
- Facebook cover: use `facebook-cover.png`, 820 x 312.
- YouTube avatar: use `social-avatar.png`; upload sites can downsample from 1080 x 1080.
- YouTube cover: use `youtube-cover.png`, 2560 x 1440.
- Instagram profile: use `instagram-profile.png`, 1080 x 1080.
- Pinterest profile: use `pinterest-profile.png`, 1080 x 1080.
